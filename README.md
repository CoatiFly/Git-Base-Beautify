# Git-Base-Beautify

此项目为 git base 界面颜色修改美化用途，参考了 https://github.com/mavnn/mintty-colors-solarized 项目。


## Git Base defaults

Git Bash：

	 cd ~
	 vi .minttyrc

就可以开始编辑它的配置文件。

	ForegroundColour= ...
	BackgroundColour= ...
	CursorColour= ...

其他参数可参照项目.minttyrc文件内容。

## Replace config file

.minttyrc为git base的默认配置文件。

项目提供了两种配色方案。

只需要将文件名称修改为.minttyrc就可以了。
